# Notus

**AI-powered notes app. Everything runs locally.**

Write notes, attach images, organize with tags. Ask AI to edit, analyze, and transform your content — all without leaving the app.

## Download

| Platform | Link |
|----------|------|
| macOS (Apple Silicon) | [Notus-1.0.0-arm64.dmg](https://github.com/wozdev8/notus/releases) |
| Windows | Coming soon |

## Requirements

- [Claude Code CLI](https://docs.anthropic.com/en/docs/claude-code) — `npm i -g @anthropic-ai/claude-code`

## How it works

1. Install Claude Code
2. Download and open Notus
3. Write notes and ask AI from the prompt bar

The AI connector starts automatically with the app. No setup needed.
