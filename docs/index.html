Setup

Up and running in 2 minutes

Two steps. No sign-up. No config.

1

Install Claude Code

The AI backend that powers Notus. One command.

npm i -g @anthropic-ai/claude-code
2

Download & open Notus

Grab the .dmg for Mac or .exe for Windows. Install, open — everything works out of the box. The AI connector starts automatically.

3

Write & ask AI

Create notes, attach images, organize with tags. Ask AI to edit, analyze, or find insights — right from the prompt bar.
